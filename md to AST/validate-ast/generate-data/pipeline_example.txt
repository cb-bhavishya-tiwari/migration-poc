================================================================================================
EXAMPLE: What the Pipeline Does
================================================================================================

INPUT (from API):
-----------------
# What's included
===
✅ Basic Membership
---

STEP 1: formatMarkdown() → rawMD
----------------------------------
# What's included

---

✅ Basic Membership

---

Changes:
- Setext heading (===) → ATX heading (#)
- Normalized spacing
- Double newlines for paragraphs

STEP 2: Parse to AST
---------------------
{
  "type": "root",
  "children": [
    {
      "type": "heading",
      "depth": 1,
      "children": [{ "type": "text", "value": "What's included" }]
    },
    {
      "type": "thematicBreak"
    },
    {
      "type": "paragraph",
      "children": [{ "type": "text", "value": "✅ Basic Membership" }]
    },
    {
      "type": "thematicBreak"
    }
  ]
}

STEP 3: AST → markdown → ASTtoMD
---------------------------------
# What's included

***

✅ Basic Membership

***

Changes:
- Horizontal rule (---) → (***) [remark-stringify default]
- Trailing newline added

OUTPUT in JSON:
---------------
{
  "pricingPageId": "01JXJE4HENFW7A51F6QP80G4QY",
  "features": {
    "feat1": {
      "rawMD": "# What's included\n\n---\n\n✅ Basic Membership\n\n---",
      "ASTtoMD": "# What's included\n\n***\n\n✅ Basic Membership\n\n***\n"
    }
  }
}
